# Results Layout

The result tree is grouped by experiment family:

- `mg_core/`: early refined-MG toy and ablation artifacts.
- `mg_recall_tasks/`: delayed-recall and associative-recall artifacts.
- `glyph_memory_arena/`: glyph binding, selective-reach, v2 reduced, and
  activation-boundary artifacts.
- `viz_*`, `modal_*`: geometric-commitment visualizer and Modal artifacts used
  by the paper draft.

Most JSON result files are force-tracked because the repository ignores `*.json`
by default.
